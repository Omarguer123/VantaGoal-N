# VantaGoal World Search-Rank Pro

Premium dashboard website + SEO ranking simulator + Vercel API route.

## What is included
- Dashboard design similar to the reference style.
- Static crawlable pages for every match.
- Homepage, schedule, live scores, watch guide, teams, news, blog pages and group pages.
- Natural match-day keywords on every page.
- FAQ and SportsEvent structured data on match pages.
- `seo-system.html` and `seo-report.json` ranking simulator.
- Vercel API route: `api/football.js`.

## Vercel API variable
Add your key in Vercel:
`API_FOOTBALL_KEY`

Do not put the key inside public HTML or JavaScript.

## After publishing
Replace `https://example.com` with your real domain in canonical URLs and sitemap.
Submit `sitemap.xml` in Google Search Console.
